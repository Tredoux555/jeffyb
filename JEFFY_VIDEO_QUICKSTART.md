## STEP 5: EDIT IT ALL TOGETHER (1 hour)

### Use: [CapCut](https://capcut.com) (Free - web or app)

**Assembly:**
1. Import all video clips
2. Arrange in order (Scene 1-9)
3. Add transitions between clips (0.3 sec crossfade)
4. Add voiceover track - sync to visuals
5. Add background music (turn volume down to 20%)
6. Add sound effects at key moments
7. Export as MP4 (1080p)
